HTML

  • The head element contains information about the webpage (aka behind the scenes)
  • The body element represents the visible content shown to the user (aka on stage in the spotlight)
  • HTML provides sturcture and framework on a webpage, to make it easy for a user to navigate
  • Is like a blank puzzle

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Colours in CSS can be assigned by semantic refrences such as blue or by a hex code.
  • Is the picture printed on to the puzzle

Git

  • Every git command must start with git
  • The command git status reveals the branch you are currently on

JavaScript

  • Is all the different puzzle pieces that link with eachother
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
